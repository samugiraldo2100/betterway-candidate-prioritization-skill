# Evaluation Execution Workflow

Use this workflow after local input files have been inventoried and readable text has been extracted.

## Purpose

The evaluation execution workflow prepares a local workspace where job descriptions and candidate profiles can be compared in a structured way.

This phase does not automatically decide who should be hired.

It prepares the files and matrix needed for evidence-based evaluation.

## Required Previous Steps

Before running this workflow, complete:

1. Local input preparation using `references/local-input-processing.md`.
2. Local text extraction using `references/local-text-extraction.md`.
